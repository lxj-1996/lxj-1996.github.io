    • css3变量切换皮肤

    核心思路是利用css的类选择器，css变量，通过按钮出发点击事件切换root 根元素的属性值，从而切换变量，改变界面皮肤颜色

    1:写个主题变量色、

    :root

    css伪类匹配文档的根元素，对于html来时 :root表示元素除了优先级更高之外，与 html 选择器相同。所有主流浏览器均支持 :root 选择器，除了 IE8 及更早的版本。在声明全局 CSS 变量时 :root 会很有用。

    css3变量

    –param:#c00;

    – + 变量名

    使用

    border:var(–border)

    –border:var(–border)

    不可以作为变量名使用 ——》》 var(–border)：#000
    1. css中的变量名是区分大小写的--navColor和--navcolor是两个不同的变量;
    2. 定义变量的时候:号和;中间是一个整体，整体是变量的值;
    3. 变量值也就是css的规则值，任何规则值都可以存入变量，并且变量值不用加引号;变量值也就是css的规则值，任何规则值都可以存入变量，并且变量值不用加引号;

    css的变量是有它的作用域的，在全局和布局同时定义了一个变量，会优先应用局部作用域。

    javascript中操作变量

    element.style.getPropertyValue(‘–variableName’)

    element.style.setProperty(‘–variableName’,value)

    应用变量，然后设置根元素的属性从而匹配不同的css设置样式
